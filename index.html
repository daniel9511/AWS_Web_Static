William Daniel Cortes Espitia


Soy un Ingeniero de Datos con experiencia en el diseño, desarrollo y mantenimiento de procesos de gestión de datos, especialmente en el sector financiero. Mi especialización radica en crear algoritmos de automatización utilizando Python y Talend para procesos ETL, y complementar sistemas RPA con Power Automate. He liderado proyectos de integración de datos utilizando tecnologías como SQL Server, Python (incluyendo PySpark, Pandas, Scikit-learn, requests, obdc y Opencv) y Power BI. Además, poseo la certificación AWS Cloud Practitioner de Platzi y estoy en constante formación en Data Engineering. Mi experiencia incluye el uso de metodologías ágiles y herramientas como GitHub, Git, Azure, AWS y Power BI para la gestión eficiente de bases de datos y el análisis de datos. Mi enfoque está en obtener resultados potenciando habilidades para trabajar en equipo, lo que me prepara para contribuir al éxito de proyectos dentro de la organización.
